Below are some useful tools to make resourcing a bit easier

Batch Builder Ticket Mixer
Tool Name Description Link
Batch Builder Just a simple batch maker, for the times when we need to make a quick batch to get rid of some tickets. It will appear in the same place as the ticket mixer. It allows you to choose between letters and numbers and the batch size. (This will just create a long batch that can either be pasted straight into a teams chat or into the excel sheet, it wont however arrange the copied batches into columns like the excel batcher tool does. So for bigger batches use the excel tool). Open
Ticket Mixer Allows us to mix up tickets from multiple pools before batching them so that agents are not just getting chunks of one ticket pool. Open
